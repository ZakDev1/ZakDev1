## About

Full-stack developer building open source developer tools. I work with Next.js, Node.js, TypeScript, and Expo/React Native.

## Projects

**[Patchboard](https://patchboard.vercel.app)** - Track and review dependency updates across your GitHub repos. Approve outdated packages and raise a single PR with all changes in one go. Built with Next.js 15, Supabase, and the GitHub API.

**[expo-audit](https://github.com/ZakDev1/expo-audit)** - Static analysis CLI for Expo projects. Validates app.json, eas.json, permissions, and native dependencies. Monorepo with a shared rule engine and ESLint plugin wrapper.

**[expo-health](https://github.com/ZakDev1/expo-health)** - Dependency and permissions checker for Expo projects. Audits native modules, Info.plist entries, and package health.
